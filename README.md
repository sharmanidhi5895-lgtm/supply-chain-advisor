#  GenAI Supply Chain Advisor

**MBA Major Project — Nidhi Sharma**  
**Specialization:** Production and Operations Management

---

##  About This Project

An AI-powered supply chain advisor built as a practical implementation of the research paper:  
**"Impact of Generative AI in E-Commerce Supply Chain Management"**

This tool demonstrates Generative AI in action — users can type real supply chain problems and receive instant, research-backed recommendations powered by Claude AI (Anthropic).

🔗 **Live Demo:** [https://sharmanidhi5895-lgtm.github.io/supply-chain-advisor/](https://sharmanidhi5895-lgtm.github.io/supply-chain-advisor/)

---

##  Features

- **6 Specialist Modes** — General Advisor, Demand Forecasting, Disruption Response, Inventory Optimization, Last-Mile Logistics, Strategic AI Adoption
- **Live GenAI Responses** — Powered by Anthropic Claude API (real AI, not simulated)
- **Research-Backed** — System prompts grounded in primary survey data (n=80) and case studies
- **Multi-turn Conversation** — Remembers context across the session
- **Pre-built Prompts** — Example supply chain scenarios to get started instantly
